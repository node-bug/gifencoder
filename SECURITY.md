# Security Policy

## Supported Versions

The following versions of @nodebug/gifencoder are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please send an email to nodebug@outlook.com. All security vulnerabilities will be promptly addressed.

Please do not disclose security-related issues publicly until a fix has been released.

## Security Best Practices

- Keep dependencies up to date
- Review code changes before merging
- Validate and sanitize all inputs
- Use secure coding practices
- Regularly audit dependencies for vulnerabilities

## Security Updates

Security updates are released as patch versions. We recommend keeping your @nodebug/gifencoder package up to date with the latest patch version to ensure you have the latest security fixes.

## Dependency Security

This project uses the following security-conscious dependencies:

- sharp: For secure image processing
- gifenc: For secure GIF encoding
- Node.js: Regular security updates recommended
